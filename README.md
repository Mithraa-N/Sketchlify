# Sketchify Studio

A professional digital art tool that transforms photos into high-quality pencil sketches using advanced client-side image processing.

## Features

- **Pencil Sketches**: Convert any photo into a graphite-style masterpiece.
- **Creative Controls**: Adjust contrast, brightness, and sketch intensity.
- **Texture Overlays**: Apply realistic paper textures like canvas, notebook, or parchment.
- **Before/After Comparison**: Interactive slider to compare the original photo with the sketch.
- **Social Sharing**: Share your creations directly to popular platforms.
- **100% Client-Side**: All processing happens in your browser. No data is ever uploaded to a server.

## Technologies Used

- **Vite**: Ultra-fast frontend build tool.
- **React**: Modern library for building user interfaces.
- **TypeScript**: Type-safe development for robust code.
- **Tailwind CSS**: Utility-first CSS framework for beautiful designs.
- **Shadcn UI**: High-quality accessible components.
- **Framer Motion**: Smooth animations and transitions.

## Getting Started

Follow these steps to run the project locally:

1. **Clone the repository**:
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. **Install dependencies**:
   ```sh
   npm install
   ```

3. **Start the development server**:
   ```sh
   npm run dev
   ```
